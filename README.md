#Website Giới thiệu sản phẩm OCOP trong tỉnh Vĩnh Long
Tên Sinh Viên Thực Hiện:Đặng Thái Bảo
#Giới Thiệu:
Đây là website giới thiệu các sản phẩm OCOP của tĩnh Vĩnh Long
Website có hình ảnh sản phẩm, thông tin chi tiếc, giá bán.
#Công nghệ được sử dụng:
<li>HTML, CSS, PHP</li>
<li>MySQL</li>
<li>Xampp</li>
<li>Bootstrap 5</li>
<li>Myphpadmin</li>
#Các chức năng của Website:
<li>Hiển thị danh sách sản phẩm OCOP</li>
<li>Thêm-sửa-xóa sản phẩm OCOP</li>
<li>Tìm kiếm, lọc sản phẩm</li>
#Cơ sở dữ liệu:
<li>Database: "db_website_ocop"</li>
#Hướng dẫn chạy project:
1.Cài đặt XAMPP
2.Chạy Apache và MySQL
3.Import file db_website_ocop.sql vào phpMyAdmin
Copy thư mục project "webocop" vào "htdocs"
Truy cập: localhost/traicay
